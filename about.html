About me:

Focused and driven individual re-envisioning his life as a full stack web-developer.

As part of my front-end training, I have built web applications that are "mobile first" designed using Bootstrap/HTML5/CSS3 as well as with responsive media queries. Additionally, I have also built single page applications using JavaScript to query API’s and dynamically manipulate DOM content.

On the server side, I have experience with building small-scale API's using a "RESTful" approach and CRUD operations using NodeJS, expressJS, and NPM. My database training has started using mySQL databases. Outside of my class studies, my interests also include 3D animations using CSS and threeJS.
